# Movies API Documentation

REST API for managing movies

## Base URL
```
http://localhost:8080/movies
```

## Endpoints

### 1. Get All Movies
Retrieves all movies

**Endpoint:** `GET /movies`

**Response:**
```json
[
  {
    "id": 1,
    "title": "title1",
    "director": "director1",
    "year": 2015
  },
  {
    "id": 2,
    "title": "title2",
    "director": "director2",
    "year": 2015
  }
]
```

**Status Codes:**
- `200 OK` - Successful retrieval

---

### 2. Get Movie by ID
Retrieves a specific movie by ID

**Endpoint:** `GET /movies/{id}`

**Path Parameters:**
- `id` (integer) - The movie ID

**Example Request:**
```
GET /movies/1
```

**Response:**
```json
{
  "id": 1,
  "title": "title1",
  "director": "director1",
  "year": 2015
}
```

**Status Codes:**
- `200 OK` - Successful retrieval
- `404 Not Found` - Movie not found

---

### 3. Create Movie
Creates a new movie

**Endpoint:** `POST /movies`

**Headers:**
```
Content-Type: application/json
```

**Request Body:**
```json
{
  "title": "The Matrix",
  "director": "Wachowski Sisters",
  "year": 1999
}
```

**Note:** The `id` is automatically generated by the backend

**Status Codes:**
- `200 OK` - Successful creation

---

### 4. Update Movie
Updates an existing movie

**Endpoint:** `PUT /movies/{id}`

**Path Parameters:**
- `id` (integer) - The ID of the movie to update

**Headers:**
```
Content-Type: application/json
```

**Request Body:**
```json
{
  "title": "The Matrix Reloaded",
  "director": "Wachowski Sisters",
  "year": 2003
}
```

**Example Request:**
```
PUT /movies/1
```

**Status Codes:**
- `200 OK` - Successful update

---

### 5. Delete Movie
Deletes a movie

**Endpoint:** `DELETE /movies/{id}`

**Path Parameters:**
- `id` (integer) - The ID of the movie to delete

**Example Request:**
```
DELETE /movies/1
```

**Status Codes:**
- `200 OK` - Successful deletion

---

## Movie Object Schema

```json
{
  "id": "integer (auto-generated)",
  "title": "string",
  "director": "string",
  "year": "integer"
}
```

## Initial Data
The API is initialized with 3 movies:
- Movie 1: title1, director1, 2015
- Movie 2: title2, director2, 2015
- Movie 3: title3, director3, 2015

## Testing with Postman

### Import Collection
You can create a Postman collection with all these endpoints for easy testing.

### Example Workflow
1. Get all movies - `GET /movies`
2. Create a new movie - `POST /movies`
3. Get the new movie by ID - `GET /movies/{id}`
4. Update the movie - `PUT /movies/{id}`
5. Delete the movie - `DELETE /movies/{id}`
